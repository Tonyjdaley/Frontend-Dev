window Objects, BoM's, and HTML:5

week09-notes and reading

  1. Window Objects are objects that tranlate or communicate with the browser itself,
    this can involve browser history, cookies, etc. they act like global varables for
    your browser
  2. BOM’s or browser object model is the model type for our object communicating with
    the browser, this model encompasses how to correctly create objects that can respond
    to information provided by the browser.
  3. Html5 is currently the latest version of Hyper Markup Text Language, it adds many
    more features, tags, and elements that can’t be found in html, including replacing a
    few with more functional and appropriate tags. the main difference between the two is
    that html5 has more capablities in manipulating sound files and videos

this week i gained lots of input on how html5 works and why it is important, it is also
important to note the BOM that directly helps us utilize browser information.
